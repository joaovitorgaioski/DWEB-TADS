Tags de formatação e texto

del

A tag 'del' trassa um risco no meio do texto, como se tivesse sido "deletado".

A melhor comida do mundo é feijão strogonoff.

ins

A tag 'ins' sublinha o texto.

Dessa forma.

sub

A tag 'sub' faz o texto ficar pequeno e em baixo.

Fórmula da água: H2O

sup

A tag 'sup' faz o texto ficar pequeno e em cima.

E=mc2

blockquote

A tag 'blockquote' separa citações do resto do texto.

Alguem uma vez disse:

Lágrimas não são argumentos.

q

A tag 'q' faz citações.

Disse uma vez Francis: Desligai os vossos monitores e assim foi feito.

abbr

A tag 'abbr' coloca esses pequenos pontinhos abaixo do texto. Destaca abreviações.

Fica mais ou menos assim, não é muito bonito

address

A 'tag' address é usada para colocar informações de contato.

Pedro Koppe, 100 - Vila Matilde, Irati - PR, 84507-302

cite

A tag 'cite' é usada para referenciar um trabalho artístico.

A soma dos quadrados dos catetos é igual ao quadrado da hipotenusa.

bdo

A tag 'bdo' inverte o texto.

subi no ônibus

code

A tag 'code' define um texto com aspecto de código.

Esse texto aqui fica pequeno. 1 + 1 = 2.

pre

A tag 'pre' divide o conteúdo organizadamente.

No dia em que eu sai de casa, minha mãe me disse...
Filho, vêm cá...

kbd

A tag 'kbd' é usada para indicar onde os usuários devem clicar.

Clique aqui para ficar rico
