HTML

A linguagem HTML - Hypertext Markup Language é uma linguagem de marcação de conteúdo (texto, imagens, vídeos, audios, etc).

Tags de Título

As tags de título sao definidas pela letra 'h' e possuem 6 níveis de título. Como por exemplo:

Título nível 1

Título nível 2

Título nível 3

Título nível 4

Título nível 5
Título nível 6

Tags de destaque de texto

Para realizar o destaque de um texto podemos usar o negrito ou itálico.

O negrito é definido pelas tags 'b' ou 'strong'. Exemplo:

Texto negritado com 'b'

Texto negritado com 'strong'

Itálico é dado pelas tags 'i' ou 'em'. Exemplo:

Texto em itálico com 'i'.

Texto em itálico com 'em'.

Alterações de fonte
