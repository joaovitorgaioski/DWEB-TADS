DOM

Document Object Model

É uma API para facilitar a manipulação de documentos HTML através do JavaScript. O documento HTML é visto como "document" e a ele teremos vários métodos associados.

  1. getElementById("id") - obtem o elemento pelo id

  2. innerHTML - inserir ou mudar o conteúdo

  4. createElement - cria elementos HTML

  5. remove - remove elemento HTML
